# 登録販売者 試験対策 Practical Ver10.4.1

初期パスワード: touhan2026

## Ver10.4.1追加・修正
- 問題データ管理画面を追加
- 取り込み済み問題JSONの削除に対応
- セット単位の回答履歴削除に対応
- サンプル問題をアプリ本体ZIPから除外
- Service Workerキャッシュ名を `touhan-v10-4-1` に更新

## Ver10.4.0追加・修正
- アプリ内にVersion / Build表示を追加
- Service Workerキャッシュ名を `touhan-v10-4-0` に更新
- 一問一答 / 本番形式 / セットごとの回答履歴分離を強化
- 問題未登録セットでは前セットの履歴を表示しないよう修正
- 4択は4個、5択は5個だけ表示
- 存在しない選択肢番号の回答は無効化

## Ver10.3追加
- 問題未登録セットでは回答履歴を0件にリセット
- 空セットで前セットの履歴が見える問題を修正
- 4択は4個、5択は5個だけ表示
- 選択肢数に存在しない回答値は保存時に無効化
- 採点結果JSON取り込みでも1〜5の本番形式回答に対応
